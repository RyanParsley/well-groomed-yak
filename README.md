## Usage

Once the deck is spun up, hit `c` to clone an instance for another screen and `p` to toggle presenter mode.

## Install

```
npm i
```

## Serve

```
npm start
```

## Build

```
npm run build
```

## Dependencies

This is build with Remark.js and the Remarker CLI.
